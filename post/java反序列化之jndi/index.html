jndi+log4j2

Log4j2是Apache软件基金会下的一个开源项目,是Java编程语言的一种流行的、模块化的日志记录工具,用于记录应用程序的运行状态信息和调试信息。

而在2021年11月log4j2爆出了一个代码执行漏洞,影响范围非常大,2.14.1版本及之前的都存在该漏洞

而这个漏洞的利用正好与jndi注入有关
